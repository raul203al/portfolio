# José Raúl Alpresa — Interactive Portfolio

¡Bienvenido al código fuente de mi portfolio interactivo! Este proyecto presenta mi experiencia, habilidades y proyectos personales como Desarrollador Frontend y Full-Stack, todo integrado en una interfaz moderna, completamente *responsive* y altamente interactiva usando un diseño estilo "Bento".

[Visitar Portfolio (ralpresa.dev)](https://ralpresa.dev)

## ✨ Características Principales

- **Bento Grid Layout:** Una interfaz moderna basada en mosaicos de cristal (Glassmorphism) que presenta la información de forma estructurada.
- **Temas Personalizados:** Selector dinámico de temas inspirados en distintos perfiles de color.
- **Internacionalización (i18n):** Soporte completo para múltiples idiomas (Español e Inglés).
- **Stack Tecnológico Interactivo:** Píldoras de tecnología con micro-animaciones dinámicas y atajos interactivos.
- **Diagramas de Arquitectura SVG:** Diagramas animados directamente en SVG para explicar la arquitectura de los proyectos backend (ej. WebSocket, Spring Boot).
- **Diseño Responsive:** Adaptación total y fluida desde monitores *ultrawide* hasta pantallas de teléfonos móviles.
- **Animaciones:** Un `canvas` oculto que renderiza efectos de fondo dinámicos durante la navegación.

## 🛠️ Tecnologías Utilizadas

Este proyecto ha sido desarrollado con:

- **[Angular](https://angular.dev/)** - Framework principal para la Single Page Application (SPA).
- **TypeScript** - Para asegurar tipado estricto y lógica robusta.
- **SCSS / Vanilla CSS** - Uso de variables CSS, Flexbox, Grid y animaciones personalizadas sin depender de frameworks externos como Tailwind, logrando máxima flexibilidad.
- **Devicon** - Librería de iconos vectoriales para las herramientas del stack.
- **Canvas HTML5** - Para el renderizado de efectos visuales avanzados.

## 🚀 Despliegue Local

### Requisitos Previos

- [Node.js](https://nodejs.org/) (Versión LTS recomendada)
- Angular CLI

### Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/raul203al/portfolio.git
   cd portfolio
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo local:
   ```bash
   npm run start
   # o bien
   ng serve
   ```

4. Abre tu navegador de preferencia y visita `http://localhost:4200/`.

## 🏗️ Construcción para Producción

Para compilar el proyecto optimizado para producción:

```bash
npm run build
# o bien
ng build
```

Los archivos compilados, cacheados y minificados se guardarán en el directorio `dist/`, listos para ser desplegados en cualquier servidor web, VPS o CDN.
